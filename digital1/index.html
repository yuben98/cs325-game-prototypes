Attack on Gabis

  • Originality: I help a Titan feed itself.
  • Technical Merit: I only changed elements from the original template. Not much technical merit as it this the first digital assignment.
  • Prototype Postmortem: I enjoyed playing around with elements from the template and got a bit more comfortable with Javascript.
  • Prototype Assets: I borrowed assets.

    • Gabi's head: Gabi

  • Prototype Closest Other Game: I borrowed code from this game and made some changes: https://github.com/photonstorm/phaser3-examples/blob/master/public/src/scalemanager/full%20screen%20game.js
  • High Concept: A one-sentence summary of your game.
  • Theme: Stretching. The Titan stretches as it feeds.
  • Mandated Variety: Keyboard input, deterministic, Collector, Platformer.

  • Target Audience: Attack on Titan viewers.
